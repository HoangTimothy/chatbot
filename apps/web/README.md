# Web App

Web boundary for company users.

## Planned Stack

- Next.js or React
- TypeScript
- Server-side auth session handling

## Screens

```text
Login
Workspace Dashboard
Document Upload
Document Processing Status
Chat
Citation Viewer
Admin Users
Admin Audit Logs
Retrieval Trace Viewer
```

## UX Principle

The user should not configure RAG internals.

They upload documents, ask questions, and review citations.
